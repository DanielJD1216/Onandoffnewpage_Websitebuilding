## ⚡ 초간단 시작 (경험 있는 개발자용)

Next.js에 익숙하다면, 간단히:

```bash
# 1. 의존성 설치
cd MAIN
npm install

# 2. 환경 설정
cp .env.local.example .env.local
# .env.local 파일을 편집하여 필요한 값 입력

# 3. 실행
npm run dev

# 4. Vercel에 배포
# GitHub에 푸시 → Vercel에서 import → 환경 변수 추가 → 완료!
```

**기술 스택**: Next.js 14 + TypeScript + Vercel + Tailwind + next-intl (한국어)
